## Инструкция:

1) Установка зависимостей: make install
2) Установка всех необходимых пакетов: make package-install
3) Запуск игры: make project
4) Сборка проекта: make build
5) Публикация: make publish
6) Проверка кода в соответствии с ruff: make lint

## Команды доступные в игре:

1) **go \<direction\>** Перейти в направлении (north/south/east/west)
2) **\<direction\>**    Перейти в направлении (north/south/east/west)
3) **look**	            Осмотреть текущую комнату
4) **take \<item\>**	Поднять предмет
5) **use \<item\>**	    Использовать предмет из инвентаря
6) **inventory**        Показать инвентарь
7) **solve**            Попытаться решить загадку в комнате
8) **quit или exit**    Выйти из игры
9) **help**             Показать список команд

## Как победить?:

Добраться до комнаты treasure_room и открыть сундук помощью:
1) С помощью ключа (предмет treasure_key)
2) С помощью кода (команда solve)
